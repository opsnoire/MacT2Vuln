Informations de récupération

Ce document comporte des informations sur les apps et les options disponibles dans la fonctionnalité de récupération de macOS.

Vous pouvez utiliser ces éléments pour réparer le disque interne de votre ordinateur, réinstaller une nouvelle copie de macOS tout en gardant vos fichiers et vos réglages intacts, restaurer votre système à partir d’une sauvegarde Time Machine et supprimer votre système pour réinstaller macOS.

L’utilitaire Sécurité au démarrage, l’utilitaire de réseau et Terminal sont disponibles dans le menu Utilitaires d’Utilitaires macOS. Obtenez plus d’informations sur le site Web d’assistance Apple.

Remarque : les options disponibles dans l’utilitaire Sécurité au démarrage varient selon le modèle de votre Mac. Sur l’iMac Pro, consultez À propos de l’utilitaire Sécurité au démarrage pour obtenir plus d’informations.

Se connecter à Internet

La réinstallation de macOS nécessite une connexion Internet. Pour vous connecter à Internet, vous pouvez utiliser une connexion réseau sans fil ou câblée.

Si vous n’avez pas de connexion Internet, vous pouvez toujours utiliser Utilitaire de disque pour réparer votre disque, ou utiliser Time Machine pour restaurer votre Mac à partir d’une sauvegarde.

Si l’icône du menu Wi‑Fi (en haut à droite de l’écran) affiche des barres , le Wi‑Fi est activé. Si l’icône du menu Wi‑Fi est vide , le Wi‑Fi est désactivé. Le Wi‑Fi doit être activé pour que vous puissiez rejoindre un réseau sans fil.

  • Activer le Wi‑Fi : Cliquez sur , puis sélectionnez Activer le Wi‑Fi.

Pour rejoindre un réseau sans-fil, procédez de l’une des façons suivantes :

  • Rejoindre un réseau : Cliquez sur , puis choisissez un réseau. Si nécessaire, entrez le mot de passe du réseau.
  • Rejoindre un réseau fermé : Cliquez sur , puis sélectionnez Se connecter à un autre réseau. Entrez le nom du réseau et, si nécessaire, le mot de passe correspondant.

Une fois que vous êtes connecté à Internet, vous pouvez réinstaller macOS, obtenir plus d’aide sur le site Web d’assistance Apple, et parcourir d’autres sites Web. Notez que les modules et les extensions du navigateur Safari sont désactivés.

Pour revenir à la fenêtre Utilitaires macOS, choisissez Safari > Quitter Safari.

Réparer votre disque

Si vous avez des problèmes avec votre ordinateur, ou si vous l’avez démarré et que la fenêtre Utilitaires macOS s’est affichée, vous devez peut-être réparer son disque interne.

Remarque : il est conseillé de noter les instructions qui suivent avant de fermer cette fenêtre ou de quitter Safari.

  1. Choisissez Safari > Quitter Safari.
  2. Sélectionnez Utilitaire de disque dans la fenêtre Utilitaires macOS, puis cliquez sur Continuer.
  3. Sélectionnez votre disque dans la liste située à gauche.
  4. Cliquez sur le bouton S.O.S., puis sur Exécuter, puis suivez les instructions à l’écran.

Si Utilitaire de disque n’est pas en mesure de réparer votre disque, sauvegardez autant d’informations que possible, puis suivez les instructions de la rubrique « Effacer et réinstaller macOS », que vous trouverez plus loin dans ce document.

Pour revenir à la fenêtre Utilitaires macOS, choisissez Utilitaire de disque > Quitter Utilitaire de disque.

Réinstaller macOS

Dans certains cas, il peut être nécessaire de réinstaller macOS. Cette réinstallation peut s’effectuer en gardant intacts vos fichiers et réglages utilisateur.

Remarque : il est conseillé de noter les instructions qui suivent avant de fermer cette fenêtre ou de quitter Safari.

  1. Choisissez Safari > Quitter Safari.
  2. Vérifiez que vous êtes connecté à Internet. (Pour vous connecter à un réseau sans fil, suivez les instructions de la rubrique « Se connecter à Internet » plus haut dans ce document.)

  3. Sélectionnez Réinstaller macOS, cliquez sur Continuer, puis suivez les instructions à l’écran.

    Important : dans la sous-fenêtre permettant de sélectionner un volume, sélectionnez votre volume macOS actuel (dans la plupart des cas, il s’agit du seul volume disponible).

Utiliser Time Machine pour restaurer votre système

Si avez déjà créé une sauvegarde Time Machine, utilisez Restaurer à partir d’une sauvegarde Time Machine pour rétablir l’état précédent de votre ordinateur.

Important : n’utilisez votre sauvegarde Time Machine que pour restaurer des données sur l’ordinateur qui est la source de cette sauvegarde. Si vous souhaitez transférer des données sur un nouvel ordinateur, quittez la fonctionnalité de récupération de macOS, redémarrez votre Mac et utilisez l’app Assistant migration dans macOS.

Remarque : il est conseillé de noter les instructions qui suivent avant de fermer cette fenêtre ou de quitter Safari.

  1. Choisissez Safari > Quitter Safari.
  2. Si votre sauvegarde se trouve sur une Time Capsule, vérifiez que vous êtes connecté à Internet. (Pour vous connecter à un réseau sans fil, suivez les instructions de la rubrique « Se connecter à Internet » plus haut dans ce document.)
  3. Sélectionnez Restaurer à partir d’une sauvegarde Time Machine, cliquez sur Continuer, puis suivez les instructions à l’écran.

Effacer et réinstaller macOS

Si vous effacez puis réinstallez macOS, tous vos comptes utilisateur, réglages réseau, fichiers et dossiers sont supprimés. Avant d’effacer macOS, essayez de vous reconnecter à votre système en suivant les instructions ci-dessous, puis sauvegardez tous les fichiers que vous souhaitez conserver en les copiant sur un autre disque ou appareil de stockage. Notez également les réglages réseau définis dans les préférences Réseau pour vous reconnecter plus facilement après la réinstallation de macOS.

Remarque : il est conseillé de noter les instructions qui suivent avant de fermer cette fenêtre ou de quitter Safari.

  1. Choisissez Safari > Quitter Safari.
  2. Vérifiez que vous êtes connecté à Internet. (Pour vous connecter à un réseau sans fil, suivez les instructions de la rubrique « Se connecter à Internet » plus haut dans ce document.)
  3. Sélectionnez Utilitaire de disque, puis cliquez sur Continuer.
  4. Sélectionnez votre volume dans la liste de gauche, puis cliquez sur Effacer.
  5. Nommez votre volume, choisissez APFS dans le menu contextuel Format, puis cliquez sur Effacer.
  6. Une fois le disque effacé, choisissez Utilitaire de disque > Quitter Utilitaire de disque.
  7. Sélectionnez Réinstaller macOS, cliquez sur Continuer, puis suivez les instructions à l’écran.

Quitter la fonctionnalité de récupération de macOS

Dans la plupart des cas, votre Mac redémarrera automatiquement lors de la restauration de votre système à partir de Time Machine ou de la réinstallation de macOS. Toutefois, nous vous conseillons d’essayer de vous reconnecter à votre copie existante de macOS (pour, par exemple, essayer de sauvegarder vos fichiers, tel que décrit dans la précédente section).

Remarque :il est conseillé de noter les instructions qui suivent avant de fermer cette fenêtre ou de quitter Safari.

  • Pour quitter la récupération de macOS : choisissez Redémarrer ou Éteindre dans le menu Apple.

Si vous souhaitez choisir un autre disque de démarrage avant de quitter, choisissez Disque de démarrage dans le menu Apple.
