ข้อมูลการกู้คืน

เอกสารนี้ประกอบด้วยข้อมูลเกี่ยวกับแอพและตัวเลือกที่มีให้ใช้งานได้ในการกู้คืน macOS

คุณสามารถใช้รายการเหล่านี้เพื่อซ่อมแซมดิสก์ภายในของคอมพิวเตอร์คุณ ติดตั้งสำเนาใหม่ของ macOS อีกครั้งในขณะที่ทำให้ไฟล์และการตั้งค่าผู้ใช้ของคุณคงเดิมอยู่เสมอ กู้คืนระบบของคุณจากข้อมูลสำรอง Time Machine และลบระบบของคุณ จากนั้นติดตั้ง macOS อีกครั้งได้

ยูทิลิตี้ความปลอดภัยการเริ่มต้นระบบ ยูทิลิตี้เครือข่าย และเทอร์มินัลมีให้ใช้งานในเมนูยูทิลิตี้ในยูทิลิตี้ macOS คุณสามารถค้นหาข้อมูลเกี่ยวกับคุณสมบัติเหล่านี้ได้ที่เว็บไซต์บริการช่วยเหลือของ Apple

หมายเหตุ: ตัวเลือกที่มีให้ใช้งานในยูทิลิตี้ความปลอดภัยของการเริ่มต้นระบบ จะแตกต่างกันไปตามรุ่นของ Mac ของคุณ บน iMac Pro ให้ดูข้อมูลเพิ่มเติมที่ เกี่ยวกับยูทิลิตี้ความปลอดภัยการเริ่มต้นระบบ

เชื่อมต่อกับอินเทอร์เน็ต

ในการเชื่อมต่อกับอินเทอร์เน็ต การติดตั้ง macOS อีกครั้งต้องใช้การเชื่อมต่อกับอินเทอร์เน็ต คุณสามารถใช้การเชื่อมต่อเครือข่ายแบบไร้สายและแบบผ่านสายได้

ถ้าคุณไม่ได้เชื่อมต่อกับอินเทอร์เน็ต คุณก็ยังสามารถใช้ยูทิลิตี้ดิสก์เพื่อซ่อมแซมดิสก์โวลุ่มของคุณหรือใช้ Time Machine เพื่อกู้คืน Mac ของคุณจากข้อมูลสำรองได้

ถ้าไอคอนเมนู Wi-Fi (ที่มุมขวาบนของหน้าจอ) มีแถบ แสดงว่า Wi-Fi เปิดอยู่ ถ้าไอคอนเมนู Wi-Fi ว่างเปล่า แสดงว่า Wi-Fi ปิดอยู่ คุณต้องเปิด Wi-Fi ก่อนจึงจะสามารถเข้าร่วมเครือข่ายไร้สายได้

  • เปิด Wi-Fi: คลิก แล้วเลือก เปิด Wi-Fi

ในการเข้าร่วมเครือข่ายไร้สาย ให้ทำตามขั้นตอนใดๆ ต่อไปนี้:

  • เข้าร่วมเครือข่าย: คลิก แล้วเลือกเครือข่าย ถ้าจำเป็น ให้ป้อนรหัสผ่านของเครือข่าย
  • เข้าร่วมเครือข่ายปิด: คลิก แล้วเลือกเข้าร่วมเครือข่ายอื่น ป้อนชื่อของเครือข่าย และถ้าจำเป็น ให้ป้อนรหัสผ่านของเครือข่ายด้วย

หลังจากที่คุณเชื่อมต่อกับอินเทอร์เน็ตแล้ว คุณจะสามารถติดตั้ง macOS อีกครั้ง ขอความช่วยเหลือเพิ่มเติมที่เว็บไซต์บริการช่วยเหลือของ Apple และเลือกหาเว็บไซต์อื่นๆ ได้ โปรดทราบว่าปลั๊กอินเบราว์เซอร์และส่วนขยายของ Safari จะถูกปิดใช้งาน

ในการกลับไปที่หน้าต่างยูทิลิตี้ macOS ให้เลือก Safari > ออกจาก Safari

ซ่อมแซมดิสก์โวลุ่มของคุณ

ถ้าคุณมีปัญหาเกี่ยวกับคอมพิวเตอร์ของคุณ หรือถ้าคุณเปิดคอมพิวเตอร์แล้วเห็นหน้าต่างยูทิลิตี้ macOS แสดงว่าคุณอาจต้องซ่อมแซมดิสก์ภายในของคอมพิวเตอร์ของคุณ

หมายเหตุ: คุณอาจต้องการจดขั้นตอนต่อไปนี้ก่อนที่จะปิดหน้าต่างนี้หรือออกจาก Safari

  1. เลือก Safari > ออกจาก Safari
  2. เลือกยูทิลิตี้ดิสก์ในหน้าต่างยูทิลิตี้ macOS แล้วคลิกดำเนินการต่อ
  3. เลือกดิสก์โวลุ่มของคุณในรายการทางซ้าย
  4. คลิกปุ่มการช่วยเหลือเบื้องต้น แล้วคลิกเรียกใช้ จากนั้นทำตามขั้นตอนบนหน้าจอ

ถ้ายูทิลิตี้ดิสก์ไม่สามารถซ่อมแซมดิสก์ของคุณได้ ให้สำรองข้อมูลของคุณให้มากที่สุดเท่าที่ทำได้ แล้วทำตามขั้นตอนใน “ลบแล้วติดตั้ง macOS อีกครั้ง” ในภายหลังในเอกสารนี้

ในการย้อนกลับไปที่หน้าต่างยูทิลิตี้ดิสก์ ให้เลือก ยูทิลิตี้ดิสก์ > ออกจากยูทิลิตี้ดิสก์

ติดตั้ง macOS อีกครั้ง

ในบางสถานการณ์ คุณอาจต้องติดตั้ง macOS อีกครั้ง คุณสามารถติดตั้ง macOS อีกครั้งได้ในขณะที่ทำให้ไฟล์และการตั้งค่าผู้ใช้ของคุณคงเดิมอยู่เสมอ

หมายเหตุ:คุณอาจต้องการจดขั้นตอนต่อไปนี้ก่อนที่จะปิดหน้าต่างนี้หรือออกจาก Safari

  1. เลือก Safari > ออกจาก Safari
  2. ตรวจสอบให้แน่ใจว่าคุณได้เชื่อมต่อกับอินเทอร์เน็ตแล้ว (ในการเชื่อมต่อกับเครือข่ายไร้สาย ให้ทำตามขั้นตอนใน “เชื่อมต่อกับอินเทอร์เน็ต” ซึ่งปรากฏก่อนหน้าในเอกสารนี้)

  3. เลือกติดตั้ง macOS อีกครั้ง แล้วคลิกดำเนินการต่อ จากนั้นทำตามขั้นตอนบนหน้าจอ

    สิ่งสำคัญ: ในบานหน้าต่างที่คุณเลือกดิสก์โวลุ่ม ให้เลือกดิสก์โวลุ่ม macOS ปัจจุบันของคุณ (ซึ่งส่วนใหญ่แล้ว จะมีดิสก์โวลุ่มให้เลือกเพียงดิสก์เดียว)

ใช้ Time Machine เพื่อกู้คืนระบบของคุณ

ถ้าคุณเคยสร้างข้อมูลสำรอง Time Machine ให้ใช้กู้คืนจากการสำรองข้อมูลของ Time Machine เพื่อกู้คืนทุกสิ่งทุกอย่างบนคอมพิวเตอร์ของคุณกลับเป็นสภาวะก่อนหน้านี้

สิ่งสำคัญ: ใช้เฉพาะข้อมูลสำรอง Time Machine ของคุณเพื่อกู้คืนข้อมูลไปที่คอมพิวเตอร์ที่เป็นแหล่งสำรองข้อมูล ถ้าคุณต้องการถ่ายโอนข้อมูลจาก Mac เครื่องนี้ไปที่คอมพิวเตอร์เครื่องใหม่ ให้ออกจากการกู้คืน macOS แล้วเริ่มการทำงาน Mac ของคุณใหม่ จากนั้นใช้แอพผู้ช่วยการโยกย้ายใน macOS

หมายเหตุ: คุณอาจต้องการจดขั้นตอนต่อไปนี้ก่อนที่จะปิดหน้าต่างนี้หรือออกจาก Safari

  1. เลือก Safari > ออกจาก Safari
  2. ถ้าข้อมูลสำรองของคุณอยู่บน Time Capsule โปรดตรวจสอบให้แน่ใจว่าคุณได้เชื่อมต่อกับอินเทอร์เน็ตแล้ว (ในการเชื่อมต่อกับเครือข่ายไร้สาย ให้ทำตามขั้นตอนใน “เชื่อมต่อกับอินเทอร์เน็ต” ซึ่งปรากฏก่อนหน้าในเอกสารนี้)
  3. เลือกกู้คืนจากการสำรองข้อมูลของ Time Machine แล้วคลิกดำเนินการต่อ จากนั้นทำตามขั้นตอนบนหน้าจอ

ลบและติดตั้ง macOS อีกครั้ง

ถ้าคุณลบและติดตั้ง macOS อีกครั้ง บัญชีผู้ใช้ของคุณ การตั้งค่าเครือข่าย รวมทั้งไฟล์และโฟลเดอร์ทั้งหมดของคุณจะถูกลบ ก่อนที่คุณจะลบ macOS ให้พยายามกลับเข้าสู่ระบบของคุณโดยทำตามขั้นตอนด้านล่าง จากนั้นสำรองข้อมูลไฟล์ใดๆ ที่คุณต้องการเก็บไว้โดยคัดลอกไฟล์เหล่านั้นลงในดิสก์โวลุ่มอื่นหรืออุปกรณ์จัดเก็บข้อมูลอื่น แล้วจดการตั้งค่าเครือข่ายของคุณจากการตั้งค่าเครือข่ายเพื่อทำให้การเชื่อมต่ออีกครั้งหลังจากติดตั้ง macOS อีกครั้งง่ายยิ่งขึ้น

หมายเหตุ: คุณอาจต้องการจดขั้นตอนต่อไปนี้ก่อนที่จะปิดหน้าต่างนี้หรือออกจาก Safari

  1. เลือก Safari > ออกจาก Safari
  2. ตรวจสอบให้แน่ใจว่าคุณได้เชื่อมต่อกับอินเทอร์เน็ตแล้ว (ในการเชื่อมต่อกับเครือข่ายไร้สาย ให้ทำตามขั้นตอนใน “เชื่อมต่อกับอินเทอร์เน็ต” ซึ่งปรากฏก่อนหน้าในเอกสารนี้)
  3. เลือกยูทิลิตี้ดิสก์ แล้วคลิกดำเนินการต่อ
  4. เลือกดิสก์โวลุ่มของคุณในรายการทางซ้าย แล้วคลิกปุ่มลบ
  5. ป้อนชื่อสำหรับดิสก์โวลุ่มของคุณ แล้วเลือก APFS จากเมนูรูปแบบที่แสดงขึ้น จากนั้นคลิกลบ
  6. หลังจากลบดิสก์โวลุ่มเรียบร้อยแล้ว ให้เลือกยูทิลิตี้ดิสก์ > ออกจากยูทิลิตี้ดิสก์
  7. เลือกติดตั้ง macOS อีกครั้ง แล้วคลิกดำเนินการต่อ จากนั้นทำตามขั้นตอนบนหน้าจอ

ออกจากการกู้คืน macOS

ในสถานการณ์ส่วนใหญ่ Mac ของคุณจะเริ่มการทำงานเครื่องใหม่โดยอัตโนมัติขณะกู้คืนระบบของคุณจาก Time Machine หรือขณะติดตั้ง macOS อีกครั้ง อย่างไรก็ตาม คุณอาจต้องการพยายามกลับเข้าสู่ระบบไปยังสำเนา macOS ที่มีอยู่ของคุณ (ตัวอย่างเช่น เพื่อพยายามสำรองข้อมูลไฟล์ของคุณตามที่อธิบายในส่วนก่อนหน้านี้)

หมายเหตุ: คุณอาจต้องการจดขั้นตอนต่อไปนี้ก่อนที่จะปิดหน้าต่างนี้หรือออกจาก Safari

  • ออกจากการกู้คืน macOS: เลือกเริ่มการทำงานเครื่องใหม่หรือปิดเครื่องจากเมนู Apple

ถ้าคุณต้องการเลือกดิสก์เริ่มต้นระบบอื่นก่อนออก ให้เลือกดิสก์เริ่มต้นระบบจากเมนู Apple
