恢复信息

本文稿包含 macOS 恢复中可用 App 和选项的相关信息。

您可以使用这些项目来修复电脑的内置磁盘、保持文件和用户设置不变的情况下重新安装新的 macOS 副本、从时间机器备份恢复系统以及抹掉系统并重新安装 macOS。

“启动安全性实用工具”、“网络实用工具”和“终端”位于 macOS 实用工具的“实用工具”菜单中。您可以在 Apple 支持网站上搜索关于它们的信息。

注：“启动安全性实用工具”中可用的选项因 Mac 机型而异。在 iMac Pro 上，请参阅关于启动安全性实用工具来了解更多信息。

接入互联网

重新安装 macOS 需要互联网连接。若要接入互联网，您可以使用无线或有线网络连接。

如果没有互联网连接，您仍然可以使用“磁盘工具”来修复磁盘，或使用时间机器从备份恢复 Mac。

如果 Wi-Fi 菜单图标（位于屏幕右上角）包含信号格 ，则说明 Wi-Fi 已打开。如果 Wi-Fi 菜单图标为空 ，则说明 Wi-Fi 已关闭。首先需要打开 Wi-Fi 才能加入无线网络。

  • 打开 Wi-Fi： 点按 ，然后选取“打开 Wi-Fi”。

若要加入无线网络，请执行以下任一项操作：

  • 加入网络： 点按 ，然后选取一个网络。键入网络的密码（如果需要）。
  • 加入封闭网络： 点按 ，然后选取“加入其他网络”。输入网络的名称和密码（如果需要）。

接入互联网后，您可以重新安装 macOS，访问 Apple 支持网站来获得更多帮助以及浏览其他网站。请注意，Safari 浏览器插件和扩展已停用。

若要返回“macOS 实用工具”窗口，请选取“Safari 浏览器”>“退出 Safari 浏览器”。

修复磁盘

如果您的电脑出现了问题，或者在启动电脑时看到了“macOS 实用工具”窗口，则可能需要修复电脑的内置磁盘。

注：关闭此窗口或退出 Safari 浏览器之前，不妨先记下以下说明。

  1. 选取“Safari 浏览器”>“退出 Safari 浏览器”。
  2. 在“macOS 实用工具”窗口中选择“磁盘工具”，然后点按“继续”。
  3. 在左侧列表中选择您的磁盘。
  4. 点按“急救”按钮，点按“运行”，然后按照屏幕指示操作。

如果“磁盘工具”无法修复您的磁盘，请备份尽可能多的信息，然后按照本文稿后半部分“抹掉并重新安装 macOS”中的说明操作。

若要返回“macOS 实用工具”窗口，请选取“磁盘工具”>“退出磁盘工具”。

重新安装 macOS

某些情况下，您可能需要重新安装 macOS。您可以在保持文件和用户设置不变的情况下重新安装 macOS。

注：关闭此窗口或退出 Safari 浏览器之前，不妨先记下以下说明。

  1. 选取“Safari 浏览器”>“退出 Safari 浏览器”。
  2. 确定您的电脑已接入互联网。（若要接入无线网络，请按照本文稿前半部分“接入互联网”中的说明操作。）

  3. 选择“重新安装 macOS”，点按“继续”，然后按照屏幕指示操作。

    重要事项：在您选择磁盘的面板中，选择您当前的 macOS 磁盘（大多数情况下，它是唯一可用的磁盘）。

使用时间机器恢复您的系统

如果此前已创建了时间机器备份，请使用“从时间机器备份进行恢复”将电脑上所有的内容恢复到其先前的状态。

重要事项：您的时间机器备份只能将信息恢复到作为备份来源的电脑上。如果您想要将这台 Mac 的信息传输到新电脑，请退出 macOS 恢复，重新启动 Mac，然后使用 macOS 中的“迁移助理”App。

注：关闭此窗口或退出 Safari 浏览器之前，不妨先记下以下说明。

  1. 选取“Safari 浏览器”>“退出 Safari 浏览器”。
  2. 如果您的备份在时间返回舱上，请确定您的电脑已接入互联网。（若要接入无线网络，请按照本文稿前半部分“接入互联网”中的说明操作。）
  3. 选择“从时间机器备份进行恢复”，点按“继续”，然后按照屏幕指示操作。

抹掉并重新安装 macOS

如果抹掉并重新安装 macOS，您的用户帐户、网络设置以及所有文件和文件夹会被删除。抹掉 macOS 之前，请尝试按照下方指示重新登录您的系统，然后将要保留的任何文件拷贝到其他磁盘或储存设备上进行备份。另请记下“网络”偏好设置中您网络的设置，以便在重新安装 macOS 后能轻松地重新接入网络。

注：关闭此窗口或退出 Safari 浏览器之前，不妨先记下以下说明。

  1. 选取“Safari 浏览器”>“退出 Safari 浏览器”。
  2. 确定您的电脑已接入互联网。（若要接入无线网络，请按照本文稿前半部分“接入互联网”中的说明操作。）
  3. 选择“磁盘工具”，然后点按“继续”。
  4. 在左侧列表中选择您的磁盘，然后点按“抹掉”按钮。
  5. 输入磁盘的名称，从“格式”弹出式菜单中选取“APFS”，然后点按“抹掉”。
  6. 抹掉磁盘后，选取“磁盘工具”>“退出磁盘工具”。
  7. 选择“重新安装 macOS”，点按“继续”，然后按照屏幕指示操作。

退出 macOS 恢复

大多数情况下，从时间机器恢复系统或重新安装 macOS 时，您的 Mac 会自动重启。然而，您可能需要重新登录现有的 macOS 副本（如尝试备份您的文件，上文已阐述）。

注：关闭此窗口或退出 Safari 浏览器之前，不妨先记下以下说明。

  • 退出 macOS 恢复：从苹果菜单中选取“重新启动”或“关机”。

如果想要在退出前选取其他启动磁盘，请从苹果菜单中选取“启动磁盘”。
