復原資訊

此文件包含「macOS 復原」中的 App 和可用選項的相關資訊。

您可以使用這些項目來修復電腦的內置磁碟、重新安裝新的 macOS 並保留原有的檔案和使用者設定、從「時光機」備份回復系統，以及清除系統然後重新安裝 macOS。

您可以在「macOS 工具程式」的「工具程式」選單中取用「開機安全性工具程式」、「網路工具程式」和「終端機」。也可在 Apple 支援網站上搜尋這些工具的相關資訊。

注意：可在「開機安全性工具程式」中使用的選項會因您的 Mac 機型而有所不同。在 iMac Pro 上，請參閱關於開機安全性工具程式以瞭解更多資訊。

連接 Internet

重新安裝 macOS 需要 Internet 連線。若要連接 Internet，您可以使用無線或有線網路連線。

若您沒有 Internet 連線，您仍可以使用「磁碟工具程式」來修復磁碟，或使用「時光機」備份來回復您的 Mac。

若螢幕右上角的 Wi-Fi 選單圖像包含訊號格數 ，即表示已開啟 Wi-Fi。如果 Wi-Fi 選單圖像是空的 ，表示已關閉 Wi-Fi。您必須先開啟 Wi-Fi 才能加入無線網路。

  • 開啟 Wi-Fi：按一下 ，然後選擇「開啟 Wi-Fi」。

若要加入無線網路，請執行以下操作：

  • 加入網路：按一下 ，然後選擇網路。如有需要，請輸入網路密碼。
  • 加入封閉網路：按一下 ，然後選擇「加入其他網路」。輸入網路名稱，然後輸入網路密碼（如有需要）。

連接 Internet 之後，您就可以重新安裝 macOS 了。如需更多協助，請參訪 Apple 支援網站或瀏覽其他網站。請注意，Safari 瀏覽器的外掛模組和延伸功能此時是停用的。

若要返回「macOS 工具程式」視窗，請選擇 Safari >「結束 Safari」。

修復磁碟

若電腦發生問題或開機後看見「macOS 工具程式」視窗，您可能需要修復電腦的內置磁碟。

注意：關閉此視窗或結束 Safari 前，您可能需要記下這些指示。

  1. 選擇 Safari >「結束 Safari」。
  2. 在「macOS 工具程式」視窗中選擇「磁碟工具程式」，然後按一下「繼續」。
  3. 在左側列表中選擇磁碟。
  4. 按一下「修理工具」按鈕，按一下「執行」，然後依照螢幕指示操作。

如果「磁碟工具程式」無法修復磁碟，請儘可能地備份您的資訊，然後依照此文件下方「清除並重新安裝 macOS」中的指示來執行。

若要返回「macOS 工具程式」視窗，請選擇「磁碟工具程式」>「結束磁碟工具程式」。

重新安裝 macOS

在部分情況下，可能需要重新安裝 macOS。重新安裝 macOS 時，您仍可以保留原有的檔案和使用者設定。

注意：關閉此視窗或結束 Safari 前，您可能需要記下這些指示。

  1. 選擇 Safari >「結束 Safari」。
  2. 確定已連接 Internet。（若要連接無線網路，請依照此文件上方「連接 Internet」的指示操作。）

  3. 選擇「重新安裝 macOS」，按一下「繼續」，然後依照螢幕指示操作。

    重要事項：請在選取磁碟的面板中選擇現用的 macOS 磁碟（在大部分情況下，這是唯一可選的磁碟）。

使用「時光機」回復系統

若您先前已製作「時光機」備份，即可使用「從時光機備份回復」來將電腦上的所有資料回復到先前的狀態。

重要事項：只能使用「時光機」備份將資料回復到原本備份的來源電腦上。若您要從這部 Mac 傳送資訊到新的電腦上，請結束「macOS 復原」，重新啟動 Mac，然後使用 macOS 的「系統移轉輔助程式」。

注意：關閉此視窗或結束 Safari 前，您可能需要記下這些指示。

  1. 選擇 Safari >「結束 Safari」。
  2. 如果備份位於「時光囊」，請確定已連接 Internet（若要連接無線網路，請依照此文件上方「連接 Internet」的指示操作。）
  3. 選擇「從時光機備份回復」，按一下「繼續」，然後依照螢幕指示操作。

清除並重新安裝 macOS

若您清除並重新安裝 macOS，電腦上的資料（您的使用者帳號、網路設定、所有檔案和檔案夾）都將刪除。清除 macOS 前請先依照下方指示，嘗試登入返回您的系統，將想要保留的檔案拷貝到其他磁碟或儲存裝置來進行備份。也請記下「網路」偏好設定中的網路設定，以便重新安裝 macOS 後可以再次輕鬆連線。

注意：關閉此視窗或結束 Safari 前，您可能需要記下這些指示。

  1. 選擇 Safari >「結束 Safari」。
  2. 確定已連接 Internet（若要連接無線網路，請依照此文件上方「連接 Internet」的指示操作。）
  3. 選擇「磁碟工具程式」，然後按一下「繼續」。
  4. 在左側列表中選擇磁碟，然後按一下「清除」按鈕。
  5. 輸入磁碟名稱，從「格式」彈出式選單中選擇 APFS，然後按一下「清除」。
  6. 清除磁碟後，請選擇「磁碟工具程式」>「結束磁碟工具程式」。
  7. 選擇「重新安裝 macOS」，按一下「繼續」，然後依照螢幕指示操作。

結束「macOS 復原」

在大部分情況下，從「時光機」回復系統或重新安裝 macOS 時，您的 Mac 會自動重新開機。然而，您可能會想要嘗試登入返回現有的 macOS（例如上一節提到的，嘗試備份您的檔案）。

注意：關閉此視窗或結束 Safari 前，您可能需要記下這些指示。

  • 結束「macOS 復原」：從 Apple 選單選擇「重新開機」或「關機」。

若您要在結束前選擇不同的啟動磁碟，請從 Apple 選單選擇「啟動磁碟」。
