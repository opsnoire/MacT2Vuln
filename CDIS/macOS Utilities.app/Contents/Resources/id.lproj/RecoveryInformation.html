Informasi Pemulihan

Dokumen ini berisi informasi mengenai app dan pilihan yang tersedia di Pemulihan macOS.

Anda dapat menggunakan item ini untuk memperbaiki disk internal komputer Anda, menginstal ulang salinan baru macOS dengan tetap menyimpan file dan pengaturan pengguna Anda secara utuh, memulihkan sistem Anda dari cadangan Time Machine, dan menghapus sistem Anda, lalu menginstal ulang macOS.

Utilitas Keamanan Mulai, Utilitas Jaringan, dan Terminal tersedia di menu Utilitas di Utilitas macOS. Anda dapat mencari informasi mengenai fitur tersebut di situs web Dukungan Apple.

Catatan:Pilihan yang tersedia di Utilitas Keamanan Mulai berbeda tergantung pada model Mac Anda. Di iMac Pro, lihat Mengenai Utilitas Keamanan Mulai untuk informasi lainnya.

Menghubungkan ke Internet

Penginstalan ulang macOS memerlukan koneksi internet. Untuk menghubungkan ke internet, Anda dapat menggunakan koneksi jaringan nirkabel atau berkabel.

Jika Anda tidak memiliki koneksi internet, Anda masih dapat menggunakan Utilitas Disk untuk memperbaiki disk Anda, atau gunakan Time Machine untuk memulihkan Mac Anda dari cadangan.

Jika ikon menu Wi-Fi (di pojok kanan atas layar) berisi baris , Wi-Fi dinyalakan. Jika ikon menu Wi-Fi kosong , Wi-Fi dimatikan. Wi-Fi harus dinyalakan sebelum Anda dapat bergabung dengan jaringan nirkabel.

  • Menyalakan Wi-Fi: Klik , lalu pilih Nyalakan Wi-Fi.

Untuk bergabung dengan jaringan nirkabel, lakukan salah satu hal berikut:

  • Bergabung dengan jaringan: Klik , lalu pilih jaringan. Jika perlu, ketik kata sandi jaringan.
  • Bergabung dengan jaringan tertutup: Klik , lalu pilih Bergabung Dengan Jaringan Lain. Masukkan nama jaringan dan, jika perlu, kata sandi jaringan.

Setelah Anda terhubung ke internet, Anda dapat menginstal ulang macOS, mendapatkan bantuan lainnya di situs web Dukungan Apple, dan menelusuri situs web lainnya. Ingat bahwa plug-in dan ekstensi browser Safari dinonaktifkan.

Untuk kembali ke jendela Utilitas macOS, pilih Safari > Tutup Safari.

Memperbaiki disk Anda

Jika Anda memiliki masalah dengan komputer Anda atau jika Anda memulai komputer Anda dan melihat jendela Utilitas macOS, Anda mungkin harus memperbaiki disk internal komputer Anda.

Catatan:Anda mungkin ingin mencatat instruksi berikut sebelum menutup jendela ini atau keluar dari Safari.

  1. Pilih Safari > Tutup Safari.
  2. Pilih Utilitas Disk di jendela Utilitas macOS, lalu klik Lanjutkan.
  3. Pilih disk Anda di daftar di sebelah kiri.
  4. Klik tombol Pertolongan Pertama, klik Jalankan, lalu ikuti instruksi pada layar.

Jika Utilitas Disk tidak dapat memperbaiki disk Anda, cadangkan informasi Anda sebanyak mungkin, lalu ikuti instruksi di “Menghapus dan menginstal ulang macOS” di bagian berikutnya di dokumen ini.

Untuk kembali ke jendela Utilitas macOS, pilih Utilitas Disk > Tutup Utilitas Disk.

Menginstal ulang macOS

Dalam beberapa kondisi, Anda mungkin perlu menginstal ulang macOS. Anda dapat menginstal ulang macOS dengan tetap menyimpan file dan pengaturan pengguna Anda secara utuh.

Catatan:Anda mungkin ingin mencatat instruksi berikut sebelum menutup jendela ini atau keluar dari Safari.

  1. Pilih Safari > Tutup Safari.
  2. Pastikan Anda terhubung ke internet. (Untuk terhubung ke jaringan nirkabel, ikuti instruksi di “Menghubungkan ke Internet” di bagian awal dokumen ini.)

  3. Pilih Instal Ulang macOS, klik Lanjutkan, lalu ikuti instruksi pada layar.

    Penting: Di panel tempat Anda memilih disk, pilih disk macOS Anda saat ini (dalam kebanyakan kasus, hanya satu disk yang tersedia).

Menggunakan Time Machine untuk memulihkan sistem Anda

Jika Anda sebelumnya membuat cadangan Time Machine, gunakan Pulihkan Dari Cadangan Time Machine untuk memulihkan semua yang ada di komputer Anda ke kondisi sebelumnya.

Penting: Hanya gunakan cadangan Time Machine Anda untuk memulihkan informasi ke komputer yang merupakan sumber cadangan. Jika Anda ingin mentransfer informasi dari Mac ini ke komputer baru, tutup Pemulihan macOS, mulai ulang Mac Anda, dan gunakan app Asisten Migrasi di macOS.

Catatan:Anda mungkin ingin mencatat instruksi berikut sebelum menutup jendela ini atau keluar dari Safari.

  1. Pilih Safari > Tutup Safari.
  2. Jika cadangan Anda ada di Time Capsule, pastikan Anda terhubung ke internet. (Untuk menghubungkan ke jaringan nirkabel, ikuti instruksi di “Menghubungkan ke Internet” di bagian awal dokumen ini.)
  3. Pilih Pulihkan Dari Cadangan Time Machine, klik Lanjutkan, lalu ikuti instruksi pada layar.

Menghapus dan menginstal ulang macOS

Jika Anda menghapus dan menginstal ulang macOS, akun pengguna, pengaturan jaringan, dan semua file serta folder Anda akan dihapus. Sebelum Anda menghapus macOS, coba untuk masuk kembali ke sistem Anda dengan mengikuti instruksi di bawah, lalu cadangkan file yang ingin Anda simpan dengan menyalinnya ke disk atau perangkat penyimpanan lain. Catat juga pengaturan jaringan Anda dari preferensi Jaringan agar memudahkan Anda untuk terhubung lagi setelah menginstal ulang macOS.

Catatan:Anda mungkin ingin mencatat instruksi berikut sebelum menutup jendela ini atau keluar dari Safari.

  1. Pilih Safari > Tutup Safari.
  2. Pastikan Anda terhubung ke internet. (Untuk terhubung ke jaringan nirkabel, ikuti instruksi di “Menghubungkan ke Internet” di bagian awal dokumen ini.)
  3. Pilih Utilitas Disk, lalu klik Lanjutkan.
  4. Pilih disk Anda di daftar di sebelah kiri, lalu klik tombol Hapus.
  5. Masukkan nama untuk disk Anda, pilih APFS dari menu pop-up Format, lalu klik Hapus.
  6. Setelah disk dihapus, pilih Utilitas Disk > Tutup Utilitas Disk.
  7. Pilih Instal Ulang macOS, klik Lanjutkan, lalu ikuti instruksi pada layar.

Menutup Pemulihan macOS

Di kebanyakan situasi, Mac Anda akan memulai ulang secara otomatis saat memulihkan sistem Anda dari Time Machine atau saat menginstal ulang macOS. Tetapi, Anda mungkin ingin mencoba untuk masuk kembali ke salinan macOS Anda yang ada (misalnya, untuk mencoba mencadangkan file Anda seperti yang dijelaskan di bagian sebelumnya).

Catatan:Anda mungkin ingin mencatat instruksi berikut sebelum menutup jendela ini atau keluar dari Safari.

  • Keluar dari Pemulihan macOS: Pilih Mulai Ulang atau Matikan dari menu Apple.

Jika Anda ingin memilih disk mulai yang berbeda sebelum menutup, pilih Disk Mulai dari menu Apple.
