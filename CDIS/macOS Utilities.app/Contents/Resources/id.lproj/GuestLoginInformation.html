Menghubungkan ke Internet

Sebelum Anda dapat menelusuri web, Anda harus memiliki koneksi jaringan nirkabel atau kabel. Untuk bergabung dengan jaringan nirkabel:

  1. Klik ikon status Wi-Fi di bar menu (pojok kanan atas layar), lalu pilih jaringan.
  2. Jika diperlukan, ketik kata sandi jaringan.

Untuk bergabung dengan jaringan tertutup, pilih Bergabung dengan Jaringan Lainnya. Masukkan nama jaringan dan, jika diperlukan, kata sandi jaringan.

Menelusuri Web

Setelah Anda terhubung ke Internet, Anda dapat menelusuri web di jendela Safari ini.
